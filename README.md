# routine.app




routine Pro is a smart routine tracker designed for polytechnic students .

---

##  Key Features

*   **Live Class Tracking** — Highlights the currently running class with a live indicator and countdown timer.
*   **Dual Mode** — Manage Class Routine and Daily Routine separately and switch between them with a single tap.
*   **Exam Schedule** — A dedicated section to save and manage exam routines with dates and times.
*   **Built-in Subject Database** — Includes hundreds of polytechnic subject codes and names.
*   **Cloud Backup** — Upload your routines to the cloud and restore them on any device by searching shared routines.
*   **Institute Information Setup** — Easily set your institute name, department, semester, and shift. Includes information for 40+ polytechnic institutes.
*   **Backup & Restore** — Save your routine data and transfer it to any device using the backup and restore feature.
*   **Dark / Light Theme** — Switch between dark and light modes according to your preference.
